Comment les garder sur ton téléphone

Pour pouvoir les écouter même sans connexion, où que tu sois.

📱

Sur iPhone

  1. Appuie longuement sur un bouton de téléchargement
  2. Choisis "Télécharger le fichier lié"
  3. Le fichier va dans ton app Fichiers → dossier "Téléchargements"
  4. Tu peux ensuite l'ouvrir avec Apple Music, VLC, ou n'importe quel lecteur audio/vidéo
🤖

Sur Android

  1. Touche un bouton de téléchargement
  2. Le fichier va automatiquement dans ton dossier "Téléchargements"
  3. Ouvre l'app "Mes fichiers" ou "Files" pour le retrouver
  4. Tu peux l'écouter avec n'importe quel lecteur (Google Files, VLC, etc.)
💻

Sur ordinateur

  1. Clic-droit sur un bouton → "Enregistrer le lien sous…"
  2. Choisis l'emplacement où tu veux le sauvegarder
  3. Tu peux ensuite le transférer sur ton téléphone par AirDrop, câble USB, ou cloud

💡 Une fois téléchargés, tu peux écouter tes séances même sans connexion internet, où que tu sois — dans le train, en pleine nature, dans ton lit.
